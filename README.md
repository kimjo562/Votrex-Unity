# Breakout

## How To Play

### Scenario

Move the mouse to shoot and control to the paddle. The objective is to destroy all the bricks to beat the game.

### Controls

#### Desktop

Mouse Movement - Move the Paddle, Left-Click - Shoot the Ball.

#### Mobile

Touch - Move, and shoot.
